Which model / mode should I choose?

Use the fastest/default option for simple drafting and quick questions. Use a stronger reasoning mode when the task is complex, has many constraints, or you need careful comparison. If you see options such as instant, medium or high, think of them as speed versus depth.

ModeBest forTrade-off
Instant / fastQuick wording, short emails, simple summaries, brainstorming.Fast, but less careful for complex work.
MediumMost business tasks: summaries, tables, drafts, comparisons.Good balance.
High / reasoningComplex analysis, messy files, risk review, strategy, multi-step tasks.Slower and may use more allowance.

Working with files

Ask ChatGPT to summarise, compare, extract actions, find risks, create a table, or explain the document simply. Do not upload sensitive material unless it is allowed under company policy.

Limits exist

Some tools have usage limits, especially file uploads, image generation and deep research. These limits can vary by plan and may change. If a tool stops working, try a simpler prompt, wait, or use a different tool such as Claude for long document review.
