# KGM AI Hub

A lightweight internal AI learning portal built with Astro and ready for GitHub Pages.

## Run in GitHub Codespaces

1. Upload these files to your `kgm-ai-hub` repository.
2. Open a Codespace.
3. In the terminal run:

```bash
npm install
npm run dev
```

4. Open the preview link.

## Build

```bash
npm run build
```

The built website will be in the `dist` folder.

## GitHub Pages

For GitHub Pages, keep `base: '/kgm-ai-hub'` in `astro.config.mjs` if your repo is named `kgm-ai-hub`.
