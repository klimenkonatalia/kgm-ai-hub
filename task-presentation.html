How to use this workflow

Copy one prompt, add your real context, attach a file if needed, then keep the conversation going. The first answer is rarely the final answer. Ask follow-up questions like “make it shorter”, “turn it into a table”, “focus on Australian dealers”, or “what should I verify?”.

Remember: AI cannot read your mind. If you do not explain the audience, purpose, constraints and source material, it will guess.

Dealer meeting deck

Create a 10-slide presentation outline for a dealer meeting. Context: KGM Australia, focus on sales performance, competitor activity, opportunities and next actions. For each slide include title, key message and speaker notes.
