Starter prompts

Summarise diagnostic notes

Summarise the diagnostic notes below for a technical support case.
Include: symptoms, DTCs, checks completed, likely areas to investigate, missing data and recommended next questions.
Do not invent repair instructions. Flag uncertainty clearly.

Rewrite technical explanation

Rewrite this technical explanation for a dealer technician.
Keep it accurate, structured and practical.
Use headings: Concern, Possible Cause, Checks, Required Information, Next Step.

Tip

Start small. Use AI for a first draft, a summary, a comparison table or a checklist. Then check it against your own expertise before using it.
